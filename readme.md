# Project Management Application LLD

This project represents class structures to support a Project management application

# Features
1. Represent a project using its attributes, resources and dependencies.
2. Check estimated completion scenario for a project for given set of resources.

# Models
CLASSES:
  Project
  Resource
  Task
ENUM:


# Use Cases/Assumptions
1. Different type of members contribute work hours into the project.
2. A project goes through set of stages.
3. Each stage requires set of contributions from members of multiple skills.
4. A project can depend on set of Tasks.
5. A Task uses a set of Resources
